BIG CAT EMPIRE ELEVATOR REWARD GAME

실행 방법
1. ZIP 파일을 압축 해제합니다.
2. bigcat_empire_elevator_reward_game.html을 Chrome 또는 Edge로 실행합니다.
3. 세 이미지/HTML 파일은 반드시 같은 폴더 안에 있어야 합니다.

- Level 2: Ordinal Number
- Level 3: 3·2·1 카운트다운, 문 닫히기 전 점프
- 별 보상
- 10회 콤보마다 보너스 별
